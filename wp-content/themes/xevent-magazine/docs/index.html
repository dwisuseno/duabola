Installation - top

Uploading Xevent via FTP to Wordpress Download the zipped theme pack to your local computer from themeforest and extract the ZIP file contents to a folder on your local computer. In the extracted folder you will find 'xevents.zip' Extract this file . Using an FTP client to access your host web server Upload the Theme files in the extracted folder called 'xevents' to wp-content/themes directory provided by WordPress. Visit Administration > Appearance > Themes, under the Manage Themes tab, under the Available Themes, if necessary navigate to the page displaying that Theme, then click the Activate link to make the Theme, the Current Theme..

Installing Xevent From the WordPress Dashboard - top

Download the zipped theme pack to your local computer from themeforest and extract the ZIP file contents to a folder on your local computer. Log into your WordPress admin panel and click on the “Appearance” tab. Now click on the "Add New Themes" tab. Click on the "Upload" button, "Browse" for file called 'xevents.zip' In the extracted folder select "Open". Click the "Install Now" button. Activate the theme from this page, OR "Return to Themes Page" and activate from there

General Settings - top

General Settings :to control general settings of theme as following

  • FULL WIDTH SLIDER: set to off if you want to display slider of homepage within homepage body (boxed homepage template) , note: if you select 'homepage with revolution slider' as homepage and you want to set full width to off, please make sure to set 'Force Full Width' to off in revolution slider settings.
  • RIGTH TO LEFT: Direction of theme to support right to left languages.
  • BREAKING NEWS: show/hide breaking news bar .
  • PRIMARY SIDEBAR: set wide sidebar to left,right of the page or remove it
  • SECONDARY SIDEBAR: set narrow sidebar to left,right of the page or remove it
  • FOOTER LAYOUT
  • BACKGROUND SHOW TYPE : to display as general background for theme
  • BACKGROUND VIDEO: video have to be in same server that run the Xevent theme
    • MENU SETTINGS
    • CATEGORY CONTENT DISPLAY TYPE: display content of category within the menu only if category link added to menu as a child item
    • CATEGORY ITEMS NUMBER

HOMEPAGE SETTINGS - top

Xevent homepage display general summary of the blog as following

  • RECENT POSTS: display latest posts added to blog.
  • POPULAR POSTS CATEGORY:display most visited posts of each category.
  • POSTS BY TAGS: display latest posts of most used tags in blog
  • TOP POSTS: display top 10 of best rated posts
  • LATEST VIDEO POSTS: display latest video format posts
  • AUTHOR POSTS: display latest 5 posts for each author.
  • RECENT POSTS CATEGORY: display latest posts of most used categories in blog
  • RECENT IMAGES POSTS: display latest image format posts

HOMEPAGE SETTINGS:to control homepage settings of theme as following

  • HOMEPAGE TEMPLATE:homepage with Revolution slider, homepage with Xevent slider and homepage with features posts slider
  • REVOLUTION SLIDER ALIAS: alias of revolution slider if 'homepage with Revolution slider' is selected as homepage template.
  • HOMEPAGE META:show/hide post meta data
    • AUTHOR META
    • DATE META
    • CATEGORIES META
    • COMMENTS META
    • RATE META
    • VIEWERS META
  • FEATURE POSTS:used to insert IDs of posts to display when "homepage with features posts" is selected as a homepage template

Xevent Slider settings - top

SLIDER settings:control slider of homepage when "homepage with slider" is selected as a homepage template

  • GENERAL SLIDER SETTINGS
    • EFFECT TYPE: type of transition effect between slider layers
    • SLIDER WIDTH: general width of slider in pixel
    • CONTROLE NAVIGATION : show/hide navigation for slider layers
  • LAYER SETTINGS
    • SLIDE ID :a given ID for each slider layer
    • IMAGE :a background image for slider layer
    • THUMBNAIL :a thumbnail image for slider layer
    • VIDEO LINK :an embed link of video for slider layer
    • PAUSE TIME: time in miliseconds to show each layer.
  • Elements Settings: settings of captions display into slider layer that has same given 'SLIDE ID' with their captions
    • SLIDE ID : ID of slider layer to display this element in it.
    • CAPTION: support to html,text and iframe video.
    • TRANSITION TYPE
    • ANIMATION EASING
    • TRANSITION SPEED(MILLISECONDS)
    • CAPTION X POSITION IN PIXEL: horizonal position of caption
    • CAPTION Y POSITION IN PIXEL:vertical position of caption
    • CAPTION WIDTH IN PIXEL
    • CAPTION HEIGHT IN PIXEL
    • CAPTION CLASS:custom css class to customize caption 's style that added to style.css file of Xevent theme

Article settings - top

  • POST & ARCHIVES META SETTINGS:to control posts meta data that display in single post and archive posts template.
  • RELATED POSTS SETTINGS:that display at the bottom of single post template.
  • SHARE POST SETTINGS:sharing links of social network, display in single post template.
  • OTHER SETTINGS: to DISPLAY AUTHOR INFO and NEXT/PREV ARTICLE PAGINATION in single post template.

REVIEW settings - top

REVIEW settings: control review box of Xevent as following:

  • DISPLAY REVIEW: show/hide review box.
  • REVIEW WIDGET POSITION
  • INCLUDE IN
  • EXCLUDE POST: posts IDs seperated by comma.
  • REVIEW HEADER
  • REVIEW DESCRIPTION
  • REVIEW STYLE
  • REVIEW SCALE
  • USER REVIEW
  • FEATURE NAME
  • FEATURE VALUE:float number from 1 to 5,1 to 10 or 1 to 100 as selected in 'REVIEW SCALE'.
  • REVIEW SKIN: style of review box(FONT COLOR,BACKGROUND COLOR,BAR COLOR and BAR BACKGROUND COLOR).
